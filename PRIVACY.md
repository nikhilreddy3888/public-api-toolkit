# Privacy Policy

Last updated: April 16, 2026

Public API Toolkit does not operate a hosted backend and does not collect personal data directly.

When you run the server locally, requests go from your machine to the public APIs or optional third-party providers used by the selected tool. Those providers may receive the query parameters needed to answer the request, your IP address, standard network metadata, and any API key you configured for that provider.

Optional `PUBLIC_APIS_*` keys stay in your local environment or MCP client configuration until a matching tool needs them. This project does not relay those keys anywhere else.

If you use the GitHub repository, npm package, or other hosting platforms related to this project, those services apply their own privacy policies.
