# Privacy Policy

Last updated: April 16, 2026

Public API Toolkit is an open-source plugin and MCP server that helps AI clients access structured public APIs.

## What this project collects

Public API Toolkit does not operate a hosted backend or collect personal data directly from users.

When you run the plugin locally, requests are sent from your machine to the public APIs or optional third-party providers that you choose to enable.

## Third-party services

Depending on which tools or optional API keys you configure, third-party API providers may receive:

- request parameters needed to answer the query
- your IP address and standard network metadata
- API credentials you configured locally for that provider

Those providers process data under their own privacy policies and terms.

## Local configuration

Optional API keys are stored and managed by the MCP client or local environment you use to run Public API Toolkit. This project does not transmit those keys anywhere except to the provider needed to fulfill your request.

## Repository and package distribution

If you interact with the GitHub repository, npm package, or related hosting services, those platforms may collect usage data under their own privacy policies.
