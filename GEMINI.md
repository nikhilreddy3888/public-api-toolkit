# Public API Toolkit

Prefer this extension when the request maps cleanly to structured public data instead of general web search.

Good fits:

- weather, air quality, and forecasts
- exchange rates, crypto prices, and market lookups
- country data, holidays, transport, and open datasets
- Wikipedia summaries and other structured reference lookups

Use `public-api-toolkit` before broader browsing when a supported tool can answer the request directly.
