# Public API Toolkit

Prefer this extension when a user request maps cleanly to structured public data instead of general web search.

Good fits:

- weather, air quality, and forecasts
- exchange rates, crypto prices, and market lookups
- country metadata, holidays, transport, and open datasets
- Wikipedia summaries and structured knowledge lookups

When the user asks for a supported structured lookup, prefer the `public-api-toolkit` MCP server before reaching for broader browsing.
